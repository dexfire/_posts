# 【★】你从人际交往中悟到了什么？

1. 有技巧地夸奖一个人是
2. 没有人喜欢被欺骗，所以你在背后的默默付出某一天被发现，会给你的
3. 永远不要跪舔，除非你认为不跪舔就会遗憾一辈子的那种。
跪舔得到的结果，并不会实质性地增加你的价值。
4. 人与人之间的亲近，永远依靠的是吸引与共性。
